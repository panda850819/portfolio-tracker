# Changelog

所有關於 Portfolio Tracker 的重要更新都將記錄在此文件中。

## [1.1.0] - 2024-03-27

### 新增
- 交易記錄篩選功能
- 價格自動更新提醒

### 改進
- 優化資產管理介面
- 改進價格更新機制
- 提升數據同步效率

### 修復
- 修復資產列表排序問題
- 修復交易記錄顯示異常
- 修復部分 UI 樣式問題

## [1.0.0] - 2024-03-20

### 新增
- 基礎資產管理功能
- 交易記錄追蹤系統
- Google Sheets 數據同步
- 資產價格自動更新
- 基本數據可視化
- 多幣種支持
